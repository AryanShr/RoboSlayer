This game is compiled for Linux OS only.

To run the game type ./main in the Terminal

Controls:
	Starting screen       - click anywhere on the game window to launch the game
	W                     - to jump
	hold W                - to little high jump
	A 		      - to move left
	D 		      - to move right
	left mouse click      - to shoot
	right mouse click     - for crosshair to aim
	esc or cross button   - to exit
	end Screen            - press anywhere on the game window to exit.
